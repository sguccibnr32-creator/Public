📄 概要

本論文は膜宇宙論 v4.7.8 本体の theoretical foundation を与える companion paper です。 v4.7.8 は SPARC 175 銀河、dSph 31 銀河、HSC-SSP Y3 + GAMA DR4 + KiDS の独立 3 field で Condition 15 (C15) を観測的に確立し、 MOND を p = 1.66×10⁻⁵³ で棄却しました。v4.8 はその establishment を支える閉形式体系を 11 個以上の連鎖として明示します。

2 論文同時 arXiv 投稿体制 (astro-ph.CO + hep-th cross) で、v4.7.8 (英語) は観測天文学者 / 宇宙論者を、 v4.8 (日本語) は理論物理 / 場の量子論の読者を対象としています。両者は非干渉で独立に成立します。

🔬 5 つの核心成果 (M1–M5)

M1 — FIRAS μ 歪み上限から α_PT^upper を導出

Fixsen+1996 の FIRAS μ 歪み上限 (μ < 9×10⁻⁵) に Chluba 2016 W_μ kernel を適用し、 V_ξ primary 採用で α_PT^upper = 2.96×10⁻⁵³ (c₀ = 0.83) を閉形式で導出。

M2 — τ_m bound を 5 桁幅に narrow 化

α_PT^upper から τ_m (membrane memory time) の FIRAS-only bound [1.9×10¹⁰, 1.4×10⁵³] s (40–58 桁幅) を得、Local Void (R = 22 Mpc) の因果律制約で上端を 2.26×10¹⁵ s に narrow。 合計 5 桁幅、38 桁 tighten。

M3 — NGC 3198 で 2.32% 精度の独立検証 (A 級)

N_mode の SI canonical 定義 (Eq. 2B-II) により、NGC 3198 の k_mem · N_mode product を 観測値 2.93×10³⁸ に対し予測値 2.998×10³⁸ で 2.32% PASS。foundation の最強 anchor。

M4 — χ 記号の 2 変数独立性を明示

同じ記号 χ が担う χ_E [s²] (MOND anchor 由来) と χ_F [m³/²/s²] (膜音速由来) は独立で、 universal conversion は存在しない。本 companion は誤解予防のため M4 として explicit に認める。

M5 — Pantheon+ f=0.379 は μ kernel 非寄与 (explicit null)

Scolnic+2022 の f = 0.379±0.029 は c₀ = 0.83 とは独立な宇宙論量で、μ 歪み kernel の integrand には現れない。 f ∼ 1−c₀ 等の spurious 関係の誤解を予防するため explicit null として明示。

⭐ 独立成果: universal coupling slope b_α across 3.92 dex

SPARC 124 銀河 (bridge 除外後) と dSph 30 銀河で独立に fit した coupling slope が 3.92 dex の密度範囲で 0.5% 以内に一致:

この結果は α (linear coupling) vs γ (threshold coupling) モデル弁別を ΔAIC = −2.00 で α 優位に決着し、 dark matter 粒子模型が必要としない operational 制約 「u_mem → g_obs variation efficiency ∼ 11% (amplitude) / ∼ 5.5% (power)」 を与えます。両数値は同一 b_α = 0.11 に対する基準選択差 (ρ¹ amplitude vs ρ² power、階層差 √1 段階、因子 2) であり、 第一原理的変換効率は v4.9 次版 (C3-3 τ_m 物理見積 + §7.8 C3-4 P1–P4 分岐比) で確定します。
