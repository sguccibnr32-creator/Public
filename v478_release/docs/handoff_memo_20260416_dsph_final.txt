【3. 背景・前提条件】

■ 理論の概要（v4.7.7確定版、前セッションから不変）
膜宇宙論は「暗黒物質の正体は膜の折り畳み構造が生む見かけの効果」とする独自理論。

条件15 最終形:
  gc = 0.584 x Yd^{-0.361} x sqrt(a0 x vflat^2/hR)
  R^2 = 0.607, scatter = 0.286 dex
  MOND棄却: p = 1.66e-53

自己無撞着方程式（v4.7.7確定）:
  s = 1 / (1 + exp(-Delta-U(s Q) / T_m))
  Delta-U(c) = -3/2 + 2 sqrt(c) - c/2 - (c/2) ln(c)
  ★ c > 0 全域で適用（c > 1 クランプ禁止、教訓51）
  T_m = sqrt(6), A=1.263, beta=-0.331
  T_m->inf: C15と完全一致 (eta=0.584, beta=-0.361)

膜補間関数:
  g_obs = g_bar / (1 - exp(-sqrt(g_bar/gc)))
  deep MOND 近似: gc ≈ g_obs^2 / g_bar
  正確解: gc = g_bar / (ln(1 - g_bar/g_obs))^2
  → dSphでは正確解が deep MOND より 10% (0.046 dex) 低い → 正確解を使うべき

J0-J3 条件（v4.0提案、本セッションで検証）:
  J0: c*(r) = 1 - f_p(r)  局所膜物性パラメータ
  J1: lambda_J,mem(r) = 2 c_s^2 / g_eff(r)  Jeans長（膜補正版）
  J2: M_J,mem = (4pi/3) rho (c_s^2/g_eff)^3  Jeans質量（膜補正版）
  J3: M_J,eff = min(M_J,std, M_J,mem)  有効Jeans質量

【4. ここまでの経緯】

  Wolf+2010逆解き: σ = sqrt(G Mdyn / (4 R_eff))
  R1単位判定（arcmin）正しく動作、rh中央値285 pc（妥当）
  3つの重大バグが露呈:
    Bug1: Mdyn は 10^6 Msun 単位（スクリプトは log10 と誤判定）
    Bug2: gc_obs（無次元）と gc_c15（SI）の単位不整合 → +12 dex の偽残差
    Bug3: Yd感度テストの shift が効いていない（scatter不変）

(3) 3バグ修正版 (v3スクリプト)
  Bug1: Mdyn × 1e6 補正
  Bug2: gc統一 → deep MOND抽出 gc_obs = g_obs^2/g_bar [SI]
  Bug3: compute_c15_test() 関数化、各Ydで全量再計算
  結果: N=27, Yd=2.0 → bias +1.435 dex, scatter 0.891 dex
  σ_wolf/σ_lit median = 0.787 → Wolf係数(4)とWalker係数(2.5)の差に起因

  ★重要発見: Yd感度はゼロ（scatterはYdで不変、meanのみシフト）
  → dSph の +1.4 dex オフセットは Yd では吸収不能

(4) 物理的起源の分析（チャット内理論的議論）
  3つの仮説を定量評価:
    H1: 3Dトポロジー → +0.15 dex（不足）
    H2: ビリアル係数 β_c → +0.25 dex（不足）
    H3: 膜ヒステリシス（plastic記憶保持）→ 最有力候補として検証へ
  H1+H2合計 = +0.40 dex（+1.4 dex の 29% のみ）

(5) ヒステリシス仮説検証 v1
  gc_MW = 2.55 a0, gc_M31 = 1.49 a0 をC15から計算
  σ_wolf係数補正（Walker 2.5 → gc × 2.56）適用
  Test 1: MW vs M31差 +0.971 dex（H3予測+0.188の5倍超過）→ 量的破綻
  Test 2: 距離相関 r=-0.224 (非有意)
  Test 3: 孤立系 Cetus 1個のみ → 判定不能
  Test 4: gc=gc_host bias +0.797, scatter 0.882 → 定数モデルに劣る
  Test 5 (H4): gc ∝ Mdyn^-1.02 × d^-0.70, R²=0.655, scatter 0.522
    → ただしMdyn依存は Υ_dyn のトートロジーに近い可能性

(6) ヒステリシス v2（M31拡充版）
  M31: N=3 → N=11 に拡充（Collins+2013全σ値ハードコード）
  σ_wolf係数を Walker 推定式(2.5)で修正 → ratio 0.997 (完璧)
  孤立系: Cetus, Phoenix, Tucana の3銀河を追加
  
  主要結果 (N=31):
    Test 1: MW-M31差 +0.566 dex (p=0.089非有意), Cohen's d=0.71
    Test 2: 距離相関 MW r=-0.050, M31 r=-0.191 (無相関)
    Test 3: 孤立系 mean 1.270 > 衛星 0.851 → H3予測と逆
    Test 4: H3 bias +0.705 (C15 +1.499より改善), scatter不変0.853
    Test 5: gc ∝ Mdyn^-1.02 × d^-0.70, R²=0.655, scatter 0.522

(7) J0-J3 Jeans枠組みテスト
  Q_J = sqrt(g_obs/a0) を定義。Q_J = Q_naive (完全一致)
  4モデル比較（C15_naive, SCE_naive, SCE_QJ, C15_QJ）:
    → 全モデル bias +1.53 dex, scatter 0.89 dex でほぼ同一
    → s(Q_J) ≈ 0.44-0.49 で定数的、SCE補正は効かない
  gc抽出: 膜補間正確解 vs deep MOND → 正確解が0.046 dex低い

  ★核心的発見:
    gc_obs vs g_bar/a0: slope = -0.758 (r²=0.54)
    → g_barが小さいほどgcが大 = Υ_dynの分散が支配
    トートロジー分解: 純粋トートロジー勾配-1, 観測-0.76
    → 真の物理的勾配 = +0.24（g_obs ∝ g_bar^0.12、ほぼ無相関）
    → Strigari universal relation (g_obs ≈ const) の反映

(8) 理論的結論（チャット内議論、最終版）
  C15がdSphに適用不能な根本原因を特定:

  ■ J3体制逆転:
    SPARC銀河: M_J,std ≤ M_J,mem → バリオンが膜状態を駆動（C15成立）
    dSph:      M_J,mem ≪ M_J,std → 膜が構造を設定、バリオンが追従
    → 因果の方向が逆転 → gc入力変数がΣ₀ではなく膜パラメータへ

  ■ Strigari relation = 膜Jeans条件:
    dSphの g_obs ≈ const ～ 10^{-11} m/s² は膜のJeansスケールが
    バリオン分布に依存せず一定値をとることの表現
    → gcはバリオン面密度ではなく膜自身の状態で決まる

  ■ 帰結:
    dSph用のgc公式は「gc ∝ √(a₀·G·Σ₀)」の形をとらない
    代わりに膜温度T_mや宇宙論パラメータ（a₀自体）が入力になるべき


【5. 決定事項】

■ H3（ヒステリシス単独）は棄却
  根拠3点:
  (1) Test 3で孤立系gc > 衛星gc（予測と逆）
  (2) Test 2で距離相関が完全無相関
  (3) Test 4で定数モデルにAICで負ける

■ H1+H2の寄与は +0.40 dex（+1.5 dex の 29%のみ）
  H1 (3D幾何): +0.151 dex
  H2 (ビリアル係数β_c=2 vs √3): +0.250 dex

■ C15のdSph直接適用は不可（構造的不整合）
  4モデル（C15_naive, SCE_naive, SCE_QJ, C15_QJ）すべて
  bias ≈ +1.5 dex, scatter ≈ 0.9 dex で同一
  Ydでは吸収不可（scatter不変）

■ dSph gcオフセットの根本原因 = J3体制逆転
  dSphではM_J,mem ≪ M_J,std（28/31銀河で確認）
  → バリオン分布がgcを決めない体制
  → C15の前提（バリオン駆動）が成立しない

■ gc抽出方法: 膜補間正確解を使うべき
  gc_exact = g_bar / (ln(1 - g_bar/g_obs))^2
  deep MOND近似より 0.046 dex 低い（10%過大評価を回避）

■ σ_wolf係数: Walker推定式の2.5を使用
  McConnachie MdynはWalker+2009推定式(係数2.5)で算出
  Wolf+2010(係数4)を使うと σが21%過小になる
  v2で修正済み: σ_wolf/σ_lit = 0.997 (完璧)

■ Strigari relation = 膜Jeans条件
  g_obs ≈ const ～ 10^{-11} m/s² はJ0-J3で自然に理解できる
  gc_obs vs g_bar/a0 の slope = -0.76（物理的成分 +0.24のみ）

■ 教訓（本セッション追加: 54-57）
  54: McConnachie 2012 VizieRのMdynは10^6 Msun単位。
  55: gc_obs と gc_pred の単位を必ずSI→a0正規化で統一。
  56: Mdyn→σ逆算にはMdynの元推定式の係数を使うこと
      （Wolf=4, Walker=2.5）。係数ミスはσに21%、gcに4乗で影響。
  57: dSphではgc ∝ Yd は効かない（scatter不変）。
      これはJ3体制（M_J,mem ≪ M_J,std）の帰結。

■ 否定された仮説・取り消された主張
  1. H3（ヒステリシス単独）→ 棄却（Test 3逆転、Test 2無相関）
  2. C15のdSph直接適用 → 不可（+1.5 dex系統オフセット）
  3. Q_J ≠ Q_naive → 同一（比=1.0000）
  4. SCEのQ依存補正がdSphで効く → 効かない（s≈0.44-0.49で定数的）
  5. Yd帯変換でoffset吸収 → 不可（scatter不変）
  6. gc_obs ∝ Mdyn^-1 が物理的 → Υ_dynトートロジーに近い

■ gc抽出の正確な方法
  膜補間関数: g_obs = g_bar / (1 - exp(-sqrt(g_bar/gc)))
  逆解き: gc = g_bar / (ln(1 - g_bar/g_obs))^2
  → g_obs > g_bar が前提（g_obs ≤ g_bar なら gc = NaN）
  → deep MOND近似(gc=g_obs^2/g_bar)より0.046 dex低い
