① MOND とは何か——そして何が足りなかったか

銀河の外縁部では星がニュートン力学の予測より速く回転する。1983年、ミルグロムはこれを説明するために MOND（修正ニュートン力学）を提唱した。加速度が臨界値 a₀ を下回ると重力が増幅される、という経験則だ。

MOND の補間関数（simple 形）： g_obs = (1/2)·(g_N + √(g_N² + 4·a₀·g_N)) a₀ = 1.2 × 10⁻¹⁰ m/s²（普遍定数として仮定） 強重力域（g_N ≫ a₀）：g_obs ≈ g_N（ニュートン的） 弱重力域（g_N ≪ a₀）：g_obs ≈ √(a₀·g_N)（増幅） → なぜ a₀ がこの値なのか——MOND は答えを持たない

MOND は銀河の回転曲線を驚くほど正確に説明するが、第一原理的な理論的根拠を持たない経験則だ。膜宇宙論はこの欠落を埋める。

③ MOND の解析的導出——膜の臨界条件から

弾性自由エネルギー U(ε;c) の平衡条件から、有効加速度が解析的に導出される（T-3・4章）。解析解 ε_eq の閉形式が確立され、独立検証残差 10⁻¹³ 以下を確認済み。

J3 体制逆転——根本原因の特定： 膜Jeans質量 M_J,mem = (4π/3) ρ (c_s² / g_eff)³ 標準Jeans質量 M_J,std の大小関係： SPARC（回転支持）：M_J,std ≤ M_J,mem → バリオン分布が膜状態を駆動（C15 成立） dSph（圧力支持）：M_J,mem ≪ M_J,std（28/31 銀河で確認） → 膜が構造を設定、バリオンが追従 ★ 因果の方向が逆転する → dSph では g_c は Σ₀ ではなく膜固有パラメータ（T_m, a₀）で決定

10.4 c→0 極限と Bernoulli 予測

J3 体制では結合 c（バリオン駆動）が 0 に近づき、膜は固有の熱平衡状態に入る。自己無撞着方程式 Eq.(6) の Q→0 極限は普遍的自発弾性率を与える：

c→0 極限の固有膜状態： s₀ = 1 / (1 + exp(3 / (2 T_m))) T_m = √6 で数値評価： exp(3/(2√6)) = exp(0.6124) = 1.8450 ★ s₀ = 0.3515（膜の普遍的自発弾性率） 小 Q 展開（非解析的）： s(Q) ≈ s₀ + α_s √Q + O(Q), α_s = 0.110 Bernoulli 分散（2状態系の普遍揺らぎ）： Var(s) = s₀ (1 − s₀) = 0.3515 × 0.6485 = 0.228 ★ G_Strigari = s₀(1 − s₀) a₀ = 0.228 a₀ = 2.74 × 10⁻¹¹ m/s²（Strigari 普遍加速度） dSph 用 g_c 公式： g_c,dSph(g_bar) = [s₀(1−s₀)]² a₀² / g_bar

10.6 トートロジー分離——真に非自明な予測内容

候補D gc = G²/gbar の slope = −1 は、定義 gc = gobs²/gbar と Strigari 関係 gobs ≈ const の合成的帰結にすぎない可能性がある。これを多変量回帰で分離した。

定義チェック（log g_obs ~ σ, r_h, M_bar 多変量）： slopes = (+2.000, -1.000, -0.000), R² = 1.0000 → g_obs = 3σ²/r_h の定義を完全再現（トートロジー部分） Strigari null 検定（単変量）： g_obs vs σ : slope +0.75 ± 0.33 → +2.3σ （marginal） g_obs vs r_h : slope -0.47 ± 0.14 → -3.4σ ★ 棄却 g_obs vs M_bar : slope -0.008±0.049 → -0.2σ ★ null 成立 結論： ・M_bar 独立性（null）= 膜理論の真の非トートロジー予測【A級確認】 ・r_h 依存性（有意）= dSph の σ-size 関係 σ ∝ r_h^0.27 に由来 （McConnachie 2012, Brasseur 2011 が観測的に確立） → 純粋 Strigari（g_obs = 普遍定数）は部分的に棄却 H4（gc ∝ M_dyn^-1）の正体： gc = σ^(+0.00±0.81) × r_h^(-1.51±0.37) → σ 寄与ゼロ、実体は r_h^-1.5 単独依存

真の膜予測は「gobs の Mbar 独立性」であり、0.2σ で null として確認された。「slope = -1」は定義と Strigari の合成的帰結であり独立予測ではない。rh 依存性は膜理論の mean-field 近似を越える追加物理（dSph の σ-size 関係の起源）として残る。

10.7 dSph 拡張の成果と残存課題

A級確立（v4.7.8 新規）： (i) J3 体制逆転（28/31 dSph で確認） (ii) c→0 極限固有状態 s₀ = 0.3515 の導出 (iii) Bernoulli 予測 G = 0.228 a₀ が 2 独立集団で 4-5% 一致 (iv) C15 → Strigari 連続遷移（ブリッジ銀河 4/4、4基準すべて達成） (v) gobs Mbar 独立性（0.2σ null） 開問題： (a) dSph の σ ∝ rh0.27 スケーリングの第一原理的起源 → 高次 cumulant か潮汐履歴など mean-field を越える物理が要 (b) 個別銀河公式 gc = [s₀(1−s₀)]² a₀² / gbar は散布 0.7 dex が残る（B級） (c) 極矮小 dSph（Υdyn > 100）は現サンプル外、独立解析が必要

結論

成果は6層で整理できる：

  1. MOND の解析的導出：弾性自由エネルギー U(ε;c) の平衡条件から MOND が特殊ケース（gc = a₀）として第一原理的に導出された（T-3・4章・Level A）
  2. gc 幾何平均法則の確立：gc = 0.584·Υd−0.36·√(a₀·Σdyn)。α=0.5 は深MOND極限から導出。SPARC 175銀河 + LITTLE THINGS 26銀河。scatter 0.286 dex（固有限界）。ΔAIC=−130（5章・Level A）
  3. MOND の位置づけ変更：MOND の a₀ は gc の「全銀河平均」。深MOND逆変換で MOND を p = 8×10⁻³³ で棄却
  4. κ=0 と理論簡素化（v4.7新規）：膜剛性ゼロ確定。条件14動力学を棄却。条件15のみの1層構造へ簡素化。各半径で μ(x) = x/(1+x) が独立に成立（RMS = 0.090 dex）
  5. 完全方程式系（13式）：式(5) 一般化MOND + 式(5') 幾何平均法則を含む13式体系が確立。膜ラグランジアン L ≈ U(ε;c)（勾配項不要）
  6. dSph 拡張と Bernoulli 予測（v4.7.8新規）：圧力支持系で J3 体制逆転を発見。c→0 極限の Bernoulli 分散 s₀(1−s₀) a₀ = 0.228 a₀ が Strigari 普遍加速度を第一原理導出。2 独立集団（dSph 5%・ブリッジ 4%）で再現。連続遷移をブリッジ銀河 4/4 で確認。膜理論は回転支持系と圧力支持系を統一的に記述する（Level A）
MOND は「なぜかうまくいく経験則」ではなくなった。膜の弾性エネルギーが臨界条件に達する瞬間——膜固有エネルギーとバリオン結合エネルギーが釣り合う点——に gc が存在し、その幾何平均が銀河の回転曲線を支配する。MOND の a₀ はその全銀河平均に過ぎない。矮小銀河では gc が a₀ より小さく、大質量銀河では大きい——この予測は SPARC 175銀河と LITTLE THINGS 26銀河で確認された。v4.7 では κ=0 が確定し、膜は各半径で独立に応答する局所理論として簡素化された。scatter 0.286 dex の94%は還元不能な銀河間多様性であり、同じ動力学的面密度を持つ銀河でも膜の折り畳み履歴が異なれば異なる gc を持つ。そして v4.7.8 では、圧力支持系 dSph への拡張により膜理論の射程が明らかになった——バリオン結合が消失する極限で膜は自発的熱揺らぎ s₀(1−s₀) a₀ = 0.228 a₀ を示し、これが Strigari 普遍加速度の正体である。回転支持系から圧力支持系まで、MOND 以前に統一的な説明を持たなかった一連の「普遍的加速度」は、膜の弾性-熱力学から第一原理的に導出される。
